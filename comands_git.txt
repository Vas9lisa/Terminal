git *ссылка clone* - скачивание внешнего репозитория
git status - отображение состояния рабочего репозитория
git add *файл* - добавление для отслеживания файла  гитом
git add . - добавление всех файлов\доков из рабочего репозитория для отслеживания гитом
git commit - создание "снимка" состаяния репозетория.
git commit 0-m *комментарий* - тоже самое, что и выше, только комментарий пишется сразу, без открываетия редактора
git push  - отправляет все закомитенные данные на удаленный репозиторий.
git pull - загрузки содержимого из удаленного репозитория и немедленного обновления локального репозитория в локальный
